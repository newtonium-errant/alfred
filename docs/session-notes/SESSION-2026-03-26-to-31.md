## Aftermath-Lab Learnings

### Patterns Validated
- n8n webhook + Cloudflare tunnel is a solid pattern for connecting cloud services to local Alfred
- Knowledge extraction from existing CLAUDE.md files is high-value — RRTS had 6+ months of accumulated knowledge that was locked to one project
- RxFax JWT node + pgcrypto approach confirmed as superior to RRTS patterns

### Gotchas Encountered
- Microsoft live.ca blocked basic IMAP auth — personal accounts require OAuth2, app passwords don't work
- Azure Portal app registration doesn't work with personal Microsoft accounts directly — need Entra portal or free Azure account first
- OAuth2 app set to "Personal accounts only" conflicts with n8n's `/common/` endpoint — must set to "All account types"
- GPG key for GitHub CLI apt repo needed `--dearmor` flag, raw download didn't work
- Multiline commands pasted into WSL terminal can get mangled — give one command at a time
